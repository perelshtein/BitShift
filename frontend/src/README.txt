Цель - написать аналог движка для обменников с простым управлением.
Возможности:
- Автоматический поиск курса на биржах и составление кросс-курса;
- Проверка курса на CoinMarketCap или по среднему значению;
- Лимиты для обмена можно задавать в любой валюте;
- Резервы задаются в любой валюте;
- Поддержка автовыплат и приема криптовалют через Bybit и Mexc;
- Встроенная защита от DDoS-атак + интеграция с CloudFlare для определения IP;
- Отправка банковских реквизитов пользователю, возможность задать текст письма;
- Уведомления админу в группу Telegram/на почту о новых заявках;
- Шаблоны направлений и статусов заявок;
- Партнерская программа и личный кабинет для пользов;
- Реферальная программа;
- Валидация номеров кошельков;
- Отзывы с модерацией для защиты от роботов;
- Выгрузка курсов в XML;
- Новости;
- Два независимых проекта: front и back. Пользователь может написать свою админку и веб-сайт и подключить его к back.

Это front-end для обменника, содержит админку и веб-сайт.
Backend написан на Kotlin.
Для доступа к backend запросите демо-доступ или купите лицензию на сайте https://bitshift.su

Code conventions:
* Имена классов в CSS и папок желательно писать через дефис (kebab-case);
* Имена файлов *.jsx - с большой буквы (ErrorPage.jsx). Название файла чаще всего = имени компонента.
