## [0.1.0] - 2024-12-11
### Добавлено:
- настройки в файле config.json, чтобы можно было задать backend-сервер без пересборки проекта;
- React-контекст для валют и настроек;
- форма входа;
- меню с подразделами - все пункты, кроме Отзывы, Страницы, Резервы;
- верстка страниц и форм;
- доступ ко всем страницам, кроме формы входа, только после авторизации на сервере;
- проверка кук и обновление сессии для страницы Настройки;
- отдельная страница для критических ошибок;
- вывод уведомлений для пользователя в Toast;
- функция safeFetch для запросов к серверу и централизованная обработка ошибок;
- интеграция с backend для страницы Настройки - для полей Таймаут сессии и Часовой пояс.
